# Emergency Wipe for Linux

⚠️ WARNING  
This tool permanently deletes all data.  
Use only on systems you own or are authorized to manage.

## Features
- Manual confirmation required
- 5-second cancel delay
- Uses nwipe
- Optional desktop launcher

## Usage
Run:
./emergency-wipe.sh

Type WIPE-MY-PC-NOW to continue or anything else to cancel.
